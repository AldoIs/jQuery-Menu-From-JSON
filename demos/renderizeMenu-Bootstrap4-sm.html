    Renderize a multilevel menu from a JSON schemme.

    How it works

    // Data input
var data = [/* Nested Array */];
// Renderize the Menu
$('#mymenu').renderizeMenu(data, {
    rootClass: "nav navbar-nav mr-auto",
    itemClass: 'nav-item',
    linkClass: 'nav-link',
    itemHasMenuClass: "nav-item dropdown",
    linkHasMenuClass: "nav-link dropdown-toggle",
    menuClass: "dropdown-menu",
    menuLinkClass: "dropdown-item",
    menuItemHasSubmenuClass: "dropdown",
    menuLinkHasSubmenuClass: "dropdown-item dropdown-toggle",
    submenuClass: "dropdown-menu",
});
// Apply your favorite plugin
jQuery.SmartMenus.Bootstrap.init();

    Parameters

    data

    The data parameter is nested array.

    options

    ConfigTypeDefaultDescription
    activeStringwindow.location.hrefCurrent URL
    rootClassString''Css Class for <ul> (The Root)
    itemClassString''Css Class for <li> tag
    linkClassString''Css Class for <a> tag
    menuClassString''Css Class for <ul> tag
    itemHasMenuClassString''Css Class for <li> tag (with submenu)
    linkHasMenuClassString''Css Class for <a> tag (with submenu)
    submenuClassString''Css Class for <ul> tag (with submenu)
    dropdownIconStringnullHtml code for dropdown icon
